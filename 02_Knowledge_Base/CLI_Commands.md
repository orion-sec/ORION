# Day 5 – IOC Extractor V2

## Run the script

```bash
python ioc_extractor.py
```

Runs the IOC extractor.

---

## Move into Scripts folder

```bash
cd 07_Scripts
```

Moves into the scripts directory.

---

## Move back one folder

```bash
cd ..
```

Moves back one directory.

---

## Check Git status

```bash
git status
```

Shows modified, staged and untracked files.

---

## Stage all changes

```bash
git add .
```

Stages every modified file.

---

## Commit changes

```bash
git commit -m "Day 5 - IOC Extractor V2"
```

Creates a Git commit.

---

## Push to GitHub

```bash
git push
```

Uploads commits to GitHub.
